# Contributing

Thanks for your interest in improving Analiza Movies.

## Quick start
- Fork and create a feature branch.
- Run `make dev` to install tooling.
- Run `make lint` and `make test` before opening a PR.

## Pull requests
- Keep changes focused and small.
- Include tests for new behavior or bug fixes.
- Update documentation when relevant.

## Issues
- Provide steps to reproduce and expected vs. actual behavior.
- Include logs or screenshots when helpful.
